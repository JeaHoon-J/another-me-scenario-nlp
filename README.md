# another-me-scenario-nlp
또 다른 나의 세계(Parallel World Story) 시나리오 생성 및 감성분석


# 🥷 Parallel World Story NLP Pipeline

LLM을 활용하여 **평행 세계의 스토리를 생성하고 이를 NLP 기법으로 분석하는 프로젝트**입니다.  

생성형 AI로 만들어진 텍스트를 **요약, 감성 분석, 키워드 추출**을 통해 구조적으로 해석하는 **NLP 파이프라인**을 구현했습니다.

---

# 📌 Project Core

이 프로젝트의 핵심은 **생성형 AI + NLP 분석을 결합한 파이프라인 구축**입니다.

1. LLM을 활용한 **평행세계 스토리 생성**
2. 생성된 텍스트 **자동 요약**
3. 스토리의 **감성 분석**
4. 이야기의 **핵심 키워드 추출**

이를 통해 **생성된 텍스트를 단순 생성에서 끝내지 않고 의미적으로 분석하는 NLP workflow**를 구현했습니다.

---

# 🧠 Models Used

| Task | Model |
|-----|------|
| Story Generation | `MLP-KTLim/llama-3-Korean-Bllossom-8B` |
| Text Summarization | `digit82/kobart-summarization` |
| Sentiment Analysis | `beomi/KcELECTRA-base-v2022` |
| Keyword Extraction | `KeyBERT` |
---


# 🎯 Project Goal

- 생성형 AI로 생성된 텍스트를 **구조적으로 분석하는 NLP 파이프라인 구축**
- 다양한 NLP 모델을 **하나의 workflow로 통합**
- **텍스트 생성 + 텍스트 분석을 결합한 실험적 NLP 프로젝트**
